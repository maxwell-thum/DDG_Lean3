1 Combinatorial Surfaces 8%

  • Lemma 1.10 (lem:closure_eq_down_closure)
    \[ \operatorname{Cl}(S) = \bigcup _{\sigma \in S} \mathcal{P}(\sigma ) \setminus \emptyset . \]
  • Definition 1.11 (def:link)

    The link \(\operatorname{Lk}(\sigma )\) of a simplex \(\sigma \in \mathcal{K}\) is the set of all simplices \(\tau \in \mathcal{K}\) such that \(\sigma \) and \(\tau \) are disjoint and their union is a simplex. In set-builder notation,

    \[ \operatorname{Lk}(\sigma ) \coloneqq \{ \tau \in \mathcal{K}\mid \tau \cap \sigma = \emptyset , \tau \cup \sigma \in \mathcal{K}\} . \]
  • Lemma 1.12 (lem:lk_eq_cl_st_sub_st_cl)

    For any set of simplices \(S \subseteq \mathcal{K}\),

    \[ \operatorname{Lk}(S) = \operatorname{Cl}(\operatorname{St}(S)) \setminus \operatorname{St}(\operatorname{Cl}(S)). \]
  • Definition 1.2 (def:degree)

    The degree or dimension of an (abstract) simplex with \(k+1\) vertices is \(k\).

  • Definition 1.3 (def:k_simplices)

    An (abstract) simplex of degree \(k\) is called an abstract \(k\)-simplex.

  • Definition 1.4 (def:face)

    A nonempty subset of an abstract simplex is called a face.

  • Definition 1.5 (def:subcomplex)

    Given two simplicial complexes \(\mathcal{K}\) and \(\mathcal{K}'\), we say that \(\mathcal{K}'\) is a subcomplex of \(\mathcal{K}\) if \(\mathcal{K}' \subseteq \mathcal{K}\), that is, every simplex in \(\mathcal{K}'\) is a simplex in \(\mathcal{K}\) as well.

  • Definition 1.6 (def:abstract_simplicial_k-complex)

    If the set of degrees of the simplices of an abstract simplicial complex has a maximum \(k\), then that abstract simplicial complex is said to be an abstract simplicial \(k\)-complex.

  • Definition 1.7 (def:pure_abstract_simplicial_k-complex)

    Let \(\mathcal{K}\) be an abstract simplicial \(k\)-complex. If every simplex is a face of some \(k\)-simplex, then we say \(\mathcal{K}\) is pure and call it a pure simplicial \(k\)-complex.

  • Definition 1.8 (def:star)

    The star \(\operatorname{St}(\sigma )\) of a simplex \(\sigma \) in an abstract simplicial complex \(\mathcal{K}\) is the set of all simplices in \(\mathcal{K}\) having \(\sigma \) as a face.

    The star \(\operatorname{St}(S)\) of a set \(S\) of simplices in \(\mathcal{K}\) (i.e., a subset of \(\mathcal{K}\)) is the set of all simplices in \(\mathcal{K}\) having some simplex in \(S\) as a face. Equivalently, \(\operatorname{St}(S) = \bigcup _{\sigma \in S} \operatorname{St}(\sigma )\).

  • Definition 1.9 (def:closure)

    The closure \(\operatorname{Cl}(S)\) of a set of simplices \(S \subseteq \mathcal{K}\) is the smallest (by inclusion) simplicial complex containing \(S\).
